# Protótipo de Sistema de Biblioteca com Django

## 📚 Descrição do Projeto

Este projeto consiste no desenvolvimento de um **protótipo de sistema de gerenciamento de biblioteca**, criado como Trabalho de Conclusão de Curso (TCC) do curso técnico em informática.

O sistema tem como objetivo facilitar o controle de livros, usuários e empréstimos dentro de uma biblioteca escolar ou institucional.

O projeto foi desenvolvido utilizando o framework **Django**, que permite criar aplicações web de forma organizada e escalável.

---

## 🎯 Objetivo

Desenvolver um sistema web capaz de:

* Cadastrar livros
* Cadastrar usuários
* Registrar empréstimos e devoluções
* Organizar o acervo da biblioteca
* Facilitar a consulta de livros disponíveis

---

## ⚙️ Tecnologias Utilizadas

* Python
* Django
* HTML
* CSS
* SQLite (banco de dados padrão do Django)

---

## 🏗️ Funcionalidades do Sistema

* Cadastro de livros
* Cadastro de usuários
* Controle de empréstimos
* Controle de devoluções
* Listagem de livros disponíveis
* Interface web para gerenciamento

---

## 🎓 Contexto Acadêmico

Projeto desenvolvido como **Trabalho de Conclusão de Curso (TCC)** do curso Técnico em Informática.
